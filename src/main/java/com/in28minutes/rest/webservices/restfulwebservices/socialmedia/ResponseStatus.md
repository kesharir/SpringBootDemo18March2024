## Response Status for REST API: 

- Return the correct response status
  - Resource is not found => 404
  - Server Exception => 500
  - Validation Error => 400 
- Important Response Statuses
  - 200 - Success
  - 201 - Created
  - 204 - No Content
  - 401 - Unauthorized
  - 400 - Bad Request
  - 404 - Resource Not Found 
  - 500 - Server Error
